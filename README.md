# Odin's Recipes

## Description
This is my first project for Project Odin. It's a simple website with a homepage and links to three of my favorite recipes. I created it to practice the fundamentals of web page creation and connecting my work to the internet.

## What I Learned
During this project, I learned how to use HTML to create a simple page and Git to save my progress.

### HTML Fundamentals
 **Text:** How to create headings and paragraphs.

**Lists:** How to create ordered and unordered shopping lists for ingredients and the list of steps.

**Links and Images:** How to add food photos and link the homepage to the recipe pages.

### Git and GitHub
I learned how to manage my project using the terminal. These are the main commands I practiced and mastered:
`git clone`: to download the GitHub repository to my computer.

`git add`: to prepare my files before saving them.

`git commit`: to save my progress in small chunks with a message.

`git push`: to upload all my code to GitHub so everyone can see it.
